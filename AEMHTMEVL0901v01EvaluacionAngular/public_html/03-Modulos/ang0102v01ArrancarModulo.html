Desde Javascript crearás los módulos usando el método angular.module() e indicándole una serie de parámetros.
Declaración de módulo
angular.module('miAplicacion', [ ... ], function(...){ ... })
La variable "angular" la tienes como variable global cuando cargas AngularJS, dentro tiene un objeto que estará disponible en cualquier parte de tu código
"module", que es un método del objeto "angular" y que sirve para crear el módulo.
El primer argumento es el nombre del módulo, que corresponde con el nombre de tu aplicación.
En el segundo parámetro puedes indicar una serie de módulos adicionales
El tercer parámetro es opcional y en él colocamos una función que sirve para configurar AngularJS.
